# Capture Flows

## Travel day

Prompt for only missing essentials:

- date
- origin
- destination
- miles driven
- campground or overnight location
- fuel stops
- notable events
- lessons learned

## Campground review

Capture:

- name and location
- site number if safe/private
- hookups
- access/roads
- connectivity
- noise
- dog friendliness
- overall rating
- would return?

## Fuel stop

Capture:

- date
- station/location
- odometer if available
- gallons
- total cost
- price per gallon
- notes

## Maintenance

Capture:

- date
- title
- system/equipment
- symptoms
- action taken
- cost
- status
- follow-up
